# ConsoleAndMonkeys

## Description

ConsoleAndMonkeys est une application console écrite en C# manipulant des classes pour décrire le scénario suivant : 

a. Un spectateur croise deux dresseurs de singes. Chaque dresseur a un singe.
b. Chaque singe connaît une liste de tours, qui ont chacun un nom et sont soit des tours d'acrobatie ou de musique.
c. Les dresseurs peuvent demander à leurs singe d'exécuter un tour.
d. Le spectateur applaudit lors de tours d'acrobatie ou siffle pendant les tours de musique.


L'application, lorsque lancée, reproduit la situation dans la console. Une suite de tests a également été réalisée.
